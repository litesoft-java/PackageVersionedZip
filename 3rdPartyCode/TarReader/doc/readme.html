Java Tar Package

The com.ice.tar package implements a Java tar archive InputStream. With this package, you can read tar archives using Java Input. This also allows you to combine com.ice.tar with java.util.zip to read gzip-ed tar archives, or the infamous .tar.gz files!

Developer Reference

The Developer Reference provides details on using the jtar package in your own Java programs. The information is a little limited, but it should be enough to get you started.
